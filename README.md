
# graphql-apollo

This is the implementation of GraphQL using apollo-server and graphql


## Tech Stack

**Client:** Javascript

## Installation
- Node v18.19.1
- npm init --yes && npm pkg set type="module"
- npm install @apollo/server graphql

To run the client on local machine execute, 
- node index
